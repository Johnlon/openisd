WinISD homage

Classic skin · Phase 2

Separate visual world: WinISD 0.7 grey chrome, single chart, Projects list + Signal Generator, bottom Project tab rail.
Design notes. Dark-only by choice — the product commits to an instrument-screen look, so there is no light theme here. Accent #4fb0ff, curve palette straight from utils/series.ts (SPL blue, excursion amber, impedance red, group-delay violet). Disabled/unbuilt affordances stay honest: Iso-Barik → “not modelled”, Cut list / Export → “soon”. Every region maps to an existing component in packages/ui/src/; modern-plus is re-layout, not new physics.
